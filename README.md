# Mortgage Calculator (Vite + React + TypeScript)

This project is scaffolded as a **Vite + React + TypeScript** app with:

- **Tailwind CSS** for styling
- **lucide-react** for icons
- **recharts** for charts

## Getting started

### Prerequisites

- Node.js 18+ (recommended)

### Install

```bash
npm install
```

### Run dev server

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project structure

- `src/App.tsx`: Main layout + top tab navigation (6 tabs)
- `src/components/*Calculator.tsx`: Placeholder calculators (currently display “Coming Soon”)
- `tailwind.config.js`, `postcss.config.js`, `src/index.css`: Tailwind setup
